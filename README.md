Es un sistema web seguro para la gestión de inventarios desarrollado en Python con Flask, diseñado bajo la identidad visual de la Unison para la materia de Desarrollo IV. Permite administrar productos y almacenes mediante roles de usuario diferenciados (Admin, Productos, Almacenes), garantizando la integridad de los datos con un registro de auditoría y ofreciendo una experiencia fluida gracias a su búsqueda y filtrado en tiempo real.
